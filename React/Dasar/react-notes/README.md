## How to Run Locally

1. Clone This Project
2. Open Project Directory
   ```
   cd react-notes
   ```
3. Install Depedencies
   ```
   npm install
   ```
4. Run Dev
   ```
   npm run dev
   ```
5. Open the project on your web browsers
   ```
   http://localhost:5173/
   ```
